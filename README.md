# ola-rides-analysis
# 🚖 Ola Ride Analytics Platform

An end-to-end **Data Analytics Project** that analyzes ride booking patterns, cancellations, revenue trends, and customer behavior in an Ola-like ride-hailing dataset.

This project demonstrates the use of **SQL, Python, Power BI, and Streamlit** to transform raw ride data into actionable business insights through interactive dashboards and visual analytics.

---

# 📊 Project Overview

The objective of this project is to analyze ride data and identify patterns that can help improve operational efficiency, customer experience, and revenue generation for a ride-hailing platform.

The project covers the complete analytics workflow:

Data Collection → Data Cleaning → SQL Analysis → Business Intelligence Dashboard → Interactive Web App

# 🔎 Key Business Questions Answered

This analysis answers several important business questions such as:

* What are the peak ride demand hours?
* Which vehicle types generate the most revenue?
* What percentage of rides are cancelled?
* What are the main reasons for ride cancellations?
* Which payment methods are most used?
* Which locations have the highest ride demand?

# 📊 Example Insights

Some insights from the analysis include:

* Peak ride demand occurs during evening hours.
* Mini and Prime vehicles contribute significantly to total revenue.
* A notable percentage of rides are cancelled due to driver unavailability.
* UPI and digital payments dominate ride transactions.
